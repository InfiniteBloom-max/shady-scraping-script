Step-by-Step Process

1. Discovery Phase

GET https://www.shadecoder.com/blogs Parse HTML for all <a href="./blogs/.."> links Convert relative to absolute URLs Remvoe duplicates (127 total blogs)

2. Content Extraction Phase

For each blog URL, the script:

- Fetch page with requests library - Parse with BeautifulSoup - Extract <h1> title (fallback to <h2>) - Get description from meta or first paragraph - Extract all code blocks and headings - Collect all external links

3. Analysis Phase

- Scan content for topic keywords - Tag blogs with relevant topics - Build topic-to-blog index - Create reverse mapping for search

4. Output Generation

- Save all blog data as JSON - Create searchable index file - Generate markdown knowledge base
